Cómo ponchar un cable RJ45

El ponchado de un cable RJ45 es el proceso de conectar los cables de red en un conector RJ45 para permitir la comunicación de datos. Existen diferentes configuraciones según el tipo de red que se quiera establecer.

Materiales necesarios

  • Cable UTP (categoría 5e, 6, etc.)
  • Conectores RJ45
  • Crimpadora
  • Probador de cable (opcional)

Tipos de cableado

Existen dos estándares principales de cableado: T568A y T568B. Dependiendo de cómo se combinen, se pueden obtener distintos tipos de conexión:

Tipo de conexión Configuración Uso común
B-B Ambos extremos con el estándar T568B Conexión de red directa (PC a switch o router)
A-A Ambos extremos con el estándar T568A Conexión de red directa
A-B Un extremo con T568A y el otro con T568B Cable cruzado (usado antiguamente para conectar dos PCs sin switch)

Pasos para ponchar un cable RJ45

  1. Pelar aproximadamente 2 cm del recubrimiento del cable UTP.
  2. Ordenar los hilos según el estándar deseado (T568A o T568B).
  3. Recortar los hilos a la misma longitud.
  4. Insertar los hilos en el conector RJ45 asegurándose de que cada hilo entre completamente.
  5. Usar la crimpadora para fijar los hilos en el conector.
  6. Probar la conexión con un tester de cables para verificar que el ponchado fue exitoso.

Conexión a un switch y un Access Point D-Link

Para garantizar una conexión estable y segura, se conectó el cable RJ45 ponchado a un switch y un Access Point D-Link de la siguiente manera:

  • El cable se conectó desde el router principal al switch usando un cable B-B (T568B en ambos extremos).
  • Desde el switch, se llevó otro cable B-B hacia el Access Point D-Link.
  • Se configuró la seguridad de la red inalámbrica en el AP para utilizar WPA2-PSK con una clave segura.

Pruebas de seguridad y velocidad

Para asegurar una conexión estable y rápida, se realizaron pruebas con las siguientes herramientas:

  • WiFi Acrylic: Se utilizó para analizar la intensidad de la señal WiFi y detectar posibles interferencias.
  • iPerf: Se realizó una prueba de velocidad de la conexión de red midiendo el ancho de banda disponible entre dispositivos conectados.

Los resultados mostraron una conexión estable y segura, con una velocidad adecuada para la transferencia de datos y un nivel de interferencias mínimo.
